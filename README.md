Aplikasi web yang dirancang untuk membantu umat muslim selama bulan ramadhan.
Web ini memiliki beberapa fitur utama seperti :
1. Jadwal imsakiyah yang menampilkan waktu shalat dan imsak secara realtime melalui integrasi API
2. Kumpulan doa
3. Counter zikir sebagai alat bantu untuk menghitung jumlah zikir
4. Kalkulator zakat untuk mempermudah perhitungan zakat yang harus anda keluarkan
5. To do list untuk membantu progres ibadah harian

Cara penggunaan :
1. Akses Aplikasi:
- Buka URL live proyek melalui browser.
- Pastikan perangkat terhubung ke internet agar fitur jadwal shalat dapat memuat data terbaru secara otomatis.

2. Melihat Jadwal Imsakiyah:
- Navigasikan ke bagian Imsakiyah.
- Pilih nama kota Anda pada kolom pencarian atau menu drop-down yang tersedia.
- Halaman akan secara otomatis menampilkan jadwal shalat dan imsak selama bulan ramadhan.

3. Menghitung Zakat:
- Buka bagian Kalkulator Zakat.
- Pilih jenis zakat yang ingin dihitung (Zakat Penghasilan atau Zakat Emas).
- Masukkan nominal gaji bulanan atau berat emas yang Anda miliki.
- Klik tombol "Hitung Zakat" untuk melihat apakah harta Anda sudah mencapai nisab serta jumlah zakat yang wajib dibayarkan.

4. Fitur Ibadah Harian:
- Gunakan fitur Doa Ramadhan untuk membaca doa-doa penting selama bulan suci.
- Manfaatkan  Counter Dzikir digital untuk membantu menghitung jumlah bacaan dzikir Anda dengan menekan tombol tambah yang tersedia.
- Pantau progres harian Anda melalui To-Do List atau target ibadah harian agar tetap produktif selama Ramadhan.
